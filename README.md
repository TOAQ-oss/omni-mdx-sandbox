# Omni-MDX Sandboxes

This repository contains test implementations for the Omni-MDX ecosystem.

## 🚀 Projects

- **/next**: Demonstration of dynamic rendering with React and Next.js.
- **/python**: Headless integrations for data processing, AI, and desktop interfaces.

## 🛠️ Installation

### Next.js
```bash
cd next
npm install
npm run dev
```

### Python
```bash
cd python
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## 🤝 Contributions
This repository is an open space for experimenting with and extending the Omni-MDX ecosystem. You can contribute in several ways:

* **Templates & Components:** Add new components (React or Python) to the samples or components folders to enrich the rendering possibilities.

* **Sandboxes:** Improve existing implementations (Next.js, PyQt) or propose new integrations (Mobile, Bots, etc.).

* **Documentation:** Share concrete use cases, particularly for AI pipelines or voice datasets.

For any structural changes or performance improvements to the parsing engine (Rust/WASM), please contribute directly to the main repository:

👉 [TOAQ-oss/omni-mdx-core](https://github.com/TOAQ-oss/omni-mdx-core)
